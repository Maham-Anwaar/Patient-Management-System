### Server
Make sure to install this package to run server
```
cd server
npm install -g ts-node
```
And then to start the server run 
```
npm start
```
Also, create a .env file in your server directory and ask the author for all the keys.

### Client
Run the following commands

```
cd client
npm i
npm run dev
```
